# BlazorJSFileES2019

Because blazor `blazor.x.js` does not support old devices, https://github.com/dotnet/maui/discussions/27327

## Use

- Download from https://github.com/Yu-Core/BlazorJSFileES2019/releases. Quoting in the project. Replace `_framework/blazor.x.js`
- Self built
	1. Fork this repo
	2. https://github.com/softprops/action-gh-release/issues/400#issuecomment-1870332450
	3. Run github Actions workflow `Build and Release aspnetcore JS`
	4. Download from releases
